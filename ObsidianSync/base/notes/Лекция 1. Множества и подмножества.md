$U := \{ a_{1} ,a_{2} ,\dots, a_{n} \}$
$n$ — мощность $U$
$n := cap \; U$
$\sim$ — равномощно

- Два множества имеют одинаковую мощность, если между ними может быть построена взаимно-однозначное (функциональное) отображение 

- *Сюръекция* (отображение на) — отображение $\phi$, которое переносит все элементы множества $A$ на на все элементы множества $B$ $(\phi(A) = B)$
- *Инъекция* (однолистное отображение) — отображение $\phi$ такое что разным прообразам соответствуют разные образы $(\phi(x_{1}) = \phi(x_{2}) \implies x_{1} = x_{2})$
- *Биекция* — отображение, которое одновременно является инъекцией и сюръекцией

- Функция есть биективное отображение. Множество образов (область значений $E$) и прообразов (область определения $D$) должно быть явно указано.
- Множество называется *счётным*, если может быть установлено взаимно-однозначное отображение (существует биекция) с натуральными числами ($U \sim \mathbb{N}$)

- Мощность несчетных множеств может быть различна. Для действительных чисел говорят, что их мощность равна континууму.

- **Теорема транзитивности счетности.** $A \sim B$, $B \sim C$ $\implies A \sim C$

Метод доказательства по индукции:
1. Формулируется некоторое утверждение. Пусть это утверждение зависит от натурального или целочисленного параметра $n$
2. Оно оказывается верным, начиная с некоторого граничного значения 0. Если из справедливости $n_{0}$ следует справедливость $n_{0} + 1$, то истинность этого утверждения доказана для всех других параметров $n > n_{0}$
