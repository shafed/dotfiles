# Числа

- $\mathbb{Q} = \left\{  \frac{m}{n}, m \in \mathbb{Z}; \; n \in \mathbb{N}  \right\}$
- $\mathbb{R} = \mathbb{Q} \cup \mathbb{I}$
- Число $x$ называется действительным, если оно может быть представлено в виде бесконечной десятичной дроби: $x = [x], x_{1},x_{2},x_{3},\dots$
	- $[x] = \max \{ n \in \mathbb{Z} \mid n\leq x \}$
	- $\{ x \} = x - [x]$
		- $0 \leq \{ x \} < 1$

y = sgn x = \frac{x}{\lvert x \rvert } =

-1, \; x < 0 \\
0, \; x = 0
\end{cases}

- **Окрестность точки $a$, лежащей на действительной оси, радиуса $\varepsilon \; (U(a, \varepsilon))$** — множество точек числовой прямой, равноудаленных от $a$ на $\varepsilon$, т.е. $U(a, \varepsilon) = U_{\varepsilon}(a) = \{ a - \varepsilon < x < a + \varepsilon \} = \{ -\varepsilon < x - a < \varepsilon \} = \{ \lvert x-a \rvert < \varepsilon \}$

$\forall \varepsilon > 0:$
![[График окрестности.svg]]

- **Окрестность точки $a$ радиуса $\varepsilon$ с выколотым центром** — множество точек $x$, для которых выполняется следующее равенство: $U(a, \varepsilon) = \{ 0 < \lvert x-a \rvert < \varepsilon \}$

- Точка $x_{0}$ называется *предельной точкой множества* $Q \in \mathbb{R}$, если для $\forall U (x_{0}, \varepsilon): \; \exists x_{1} \in Q \; (x_{1} \neq x_{0})$

> [!info] В любой окрестности предельной точки содержится бесконечное множество точек.
> - Пусть $x_{0}$ — предельная точка множества, значит для $\forall U_{\varepsilon}(x_{0}): \exists x_{1} \in U_{\varepsilon}(x_{0})$
> 	- т.к. $x_{1} \in U_{\varepsilon}(x_{0})$, то $\lvert x_{1} - x_{0} \rvert < \varepsilon$
> - Пусть $U_{\varepsilon_{1}}(x_{0})$, где  $\varepsilon_{1} = |x_{1} - x_{0}| < \varepsilon$, — новая окрестность для точки, $x_{0}$,  тогда $\exists x_{2} \in U_{\varepsilon_{1}}(x_{0})$
> - Пусть $U_{\varepsilon_{2}}(x_{0})$, где $\varepsilon_{2} = |x_{2} - x_{0}|$, — еще одна окрестность, тогда $\exists x_{3} \in U_{\varepsilon_{2}}(x_{0})$
> - И так далее $x_{1}, x_{2}, x_{3}, \dots, x_{n} \in U_{\varepsilon}(x_{0})$

# Числовая последовательность

- Говорят, что определена последовательность действительных чисел $\{ a_{n} \}$, если для любого $n = 1,2,\dots$ поставлено в соответствие число $a_{n}$, называемое $n$-ым (или общим) членом последовательности.

- $\lim_{ n \to \infty } a_{n} = m \Leftrightarrow \forall \varepsilon > 0, \; \exists N(\varepsilon) \in \mathbb{N}: \forall n > N(\varepsilon), \; a_{n} \in U(m, \varepsilon) \; или \; |a_{n} - m| < \varepsilon$
	- Число $m$ называется **пределом** последовательности $\{ a_{n} \}$, если число $a_{n}$ отличается от $m$ сколь угодно мало, начиная с некоторого номера $N$.
	- Число $m$ называется **пределом** последовательности $\{ a_{n} \}$, если для любого, сколько угодно малого, положительного числа $\varepsilon$ найдется такой номер $N$, начиная с которого члены последовательности будут попадать в окрестность точки $m$ радиуса $\varepsilon$

- Последовательность называется *сходящейся*, если она имеет конечный предел: $\exists \lim_{ n \to \infty }a_{n} = m \Leftrightarrow \{ a_{n} \} \to m$ (последовательность $\{ a_{n} \}$ сходится к числу $m$)
- Последовательность называется *расходящейся*, если она не имеет конечного предела, т.е. $\lim_{ n \to \infty } a_{n} = \infty$ или $\lim_{ n \to \infty } \bar{\exists}$ 
	- $\lim_{ n \to \infty }a_{n} = \infty \Leftrightarrow \forall M>0 \; \exists N(M) \in \mathbb{N}: \; \forall n>N(M) \; |a_{n}| > M$
	- $\lim_{ n \to \infty }a_{n}\bar{\exists} \Leftrightarrow$
		- $\forall a \in \mathbb{R} \; \exists \varepsilon>0 \; \forall N(\varepsilon) \in \mathbb{N}: \; \exists n>N(\varepsilon) \; a_{n} \not\in U_{\varepsilon}(a) \text{ или } |a_{n} - a| \geq \varepsilon$
		- $\exists M>0 \; \forall N(M) \in \mathbb{N}: \; \exists n>N(M) \; |a_{n}| > M$

- *Бесконечно большая (ББ) последовательность* — последовательность, где $\lim_{ n \to \infty } a_{n} = \infty$
- *Бесконечно малая (БМ) последовательность* — последовательность, где $\lim_{ n \to \infty } a_{n} = 0$


> [!example] Свойства пределов числовых последовательностей
> - $m$ — предельная точка множества $Q$ членов последовательности $(Q = \{ a_{n} \})$: $\lim_{ n \to \infty }a_{n} = m$.
> - вне любой окрестности точки $m = \lim_{ n \to \infty }a_{n}$ остается конечное число членов последовательности.
> - предел сходящейся последовательности единственен.
>
>> [!success]- Доказательство
>> - пусть $a \neq b: \ a = \lim_{ n \to \infty }a_{n}, \; b = \lim_{ n \to \infty }a_{n}$ и $\varepsilon = \frac{b-a}{2}$
>>	- если $a = \lim_{ n \to \infty } a_{n}$ $a_{n} \in U_{\varepsilon}(a)$, получается в $U_{\varepsilon}(b)$ — конечное число членов последовательности $a_{n}$, но т.к. $b = \lim_{ n \to \infty } a_{n}$, то в $U_{\varepsilon}(b)$ бесконечное число членов $a_{n}$. Противоречие!
> - Сходящаяся последовательность ограничена: $\exists M > 0: \; \forall n \in \mathbb{N} \; \lvert x_{n} \rvert \leq M$
> - **Арифметические свойства пределов**:
> 	- $\exists \lim_{ n \to \infty }a_{n} =a; \; \exists \lim_{ n \to \infty } b_{n} = b \implies \exists \lim_{ n \to \infty }(a_{n} + b_{n}) = a+b$
> 		- Частный случай: $b_{n} = b = const: \forall n \implies \lim_{ n \to \infty }b\cdot a_{n} = b \cdot \lim_{ n \to \infty }a_{n}$
> 	- $\exists \lim_{ n \to \infty }a_{n} =a; \; \exists \lim_{ n \to \infty } b_{n} = b \implies \exists \lim_{ n \to \infty }(a_{n} \cdot b_{n}) = a\cdot b$
> 	- $\exists \lim_{ g \to \infty }a_{n} = a; \; \exists \lim_{ n \to \infty }b_{n} = b \neq 0 \implies \exists \lim_{ n \to \infty }\left( \frac{a_{n}}{b_{n}} \right)= \frac{a}{b}$
>
