# Принцип вложенных отрезков

- **Последовательность вложенных отрезков** — бесконечная последовательность отрезков на действительной оси $[a_{1}, b_{1}], [a_{2},b_{2}], \dots, [a_{n}, b_{n}]$, где $a_{1}\leq a_{2}\leq\dots, b_{1}\geq b_{2}\geq\dots$:

![[Последовательность вложенных отрезков.png]]

- **Принцип вложенных отрезков.** Пусть $d_{n} = b_{n} - a_{n}$ — длина $n$-ого отрезка $(b_{n} > a_{n} \forall n \in \mathbb{N})$, и пусть последовательность длин $\{ d_{n} \} \to 0 \; (\lim_{ n \to \infty }=0)$. Тогда $\exists! c \in [a_{n}, b_{n}] \forall n \in \mathbb{N}$, и при этом $\lim_{ n \to \infty } a_{n} = \lim_{ n \to \infty } b_{n} = c$ (существует единственная точка $c$, общая для всех отрезков).

# Теорема Больцано-Вейерштрасса и её следствия

- **Подпоследовательность** последовательности $\{ x_{n} \}$ — любая последовательность вида $\{ y_{k} \} = \{ x_{n_{k}} \}$, где $k = 1,2,\dots, \; n_{1},n_{2},n_{3}, \dots \in \mathbb{N}, \; n_{1} < n_{2} < n_{3} < \dots$
	- Пример: $\{ x_{2k} \} = \{ 2k \}, k \in \mathbb{N}$ — подпоследовательность $\{ x_{n} \} = \{ n \}, n \in \mathbb{N}$ 
- **Частичный предел (предельная точка)** — число $\tau$ последовательности $\{ x_{n} \}$, если существует её подпоследовательность $\{ x_{n_{k}} \} \to \tau \; (\lim_{ k \to \infty } x_{n_{k}} = \tau$.
	- Пример: $\{ x_{n} \} = \{ (-1)^{n} \} = \{ -1; 1; -1; 1; \dots \}; \; \{ x_{2k} \} = \{ (-1)^{2k} \} = \{ 1;1;1;\dots \}$, $\exists \lim_{ k \to \infty } x_{2k} = 1 \implies$ 1 — частичный предел $\{ x_{n} \}$.


> [!NOTE] Если последовательность имеет конечный предел, то любая её подпоследовательность сходится к тому же пределу
> Contents

- **Следствие.** Последовательность, для которой существуют хотя бы два различных частичных предела, *расходящаяся*.
	**пример**

- **Теорема Больцано-Вейерштрасса.** Любая ограниченная последовательность содержит сходящуюся подпоследовательность. Или: любая ограниченная последовательность имеет по меньшей мере один конечный частичный предел.
	**доказательство**

- **Следствие 1.** Любая подпоследовательность ограниченной последовательности имеет частичный предел.
- **Следствие 2.** Если все частичные пределы ограниченной последовательности одинаковы и равны $a$, то она сходится к $a$, т.е. $\lim_{ n \to \infty } x_{n} = a$.
	**доказательство**
	- **Следствие 3.** Монотонная ограниченная последовательность имеет предел.
		**доказательство + пример + 2 примечания**

# Фундаментальная последовательность

- **Фундаментальная последовательность (последовательность Коши)** — $\{ x_{n} \}$, если $\forall \varepsilon > 0 \exists N = N(\varepsilon): \lvert x_{n} - x_{m} \rvert < \varepsilon \; \forall n,m > N$.
	- т.е. у фундаментальной последовательности элементы с большими номерами не могут сильно отличаться друг от друга.
- **Критерий сходимости Коши.** Последовательность сходится в том и только в том случае, если она фундаментальна. Или: $\exists \lim_{ n \to \infty } x_{n} = a \Leftrightarrow \{ x_{n} \} \text{ фундаментальна}$. Или: $\exists \lim_{ n \to \infty } x_{n} = a \Leftrightarrow \varepsilon>0 \exists N = N(\varepsilon): \lvert x_{n} - x_{m} \rvert < \varepsilon \; \forall n,m > N$.
	**доказательство**

- **Теорема о пределе промежуточной последовательности**. Пусть $\exists \lim_{ n \to \infty } y_{n} = a, \exists \lim_{ n \to \infty } z_{n} = a, y_{n} \leq x_{n} \leq z_{n} \forall n =1,2,3,\dots$, тогда $\exists \lim_{ n \to \infty } x_{n} = a$.
	**доказательство + пример + решение**

# Неравенство Бернулли

- **Лемма.** Для $\forall x > -1, n = 1,2,3,\dots: \; (1+x)^{n} \geq 1 + nx$.
