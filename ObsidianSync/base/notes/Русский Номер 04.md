// имя текущего файла, например «Русский Номер 07»
const fileName = dv.current().file.name;

// последние две цифры номера
const numStr = fileName.slice(-2);
const num    = Number(numStr);

// вычисляем предыдущий и следующий номера (циклически 01 ↔ 27)
const prev = ((num + 25) % 27 + 1).toString().padStart(2, "0");
const next = ((num % 27)  + 1).toString().padStart(2, "0");

// функция-шаблон для meta-bind-button

class: obsidian
actions:
  - type: open
    link: "[[Русский Номер ${target}]]"
\`\`\``;

// выводим кнопки
dv.paragraph(btn(prev) + '\n\n' + btn(next));

## Причастия

### С суффиксом -Т- 

> Чаще всего ударение смещается на один слог к началу слова 

- зАгнутый (загнУть)
- зАнятый (занЯть)
- нАчатый (начАть)
- прИнятый (принЯть)
- сОгнутый (согнУть)
- зАпертый (заперЕть)

### Краткие причастия

- занятА
- принятА
- снятА
- заселенА

### Действительные причастия настоящего и прошедлего времени (-Ш-, -ВШ-)

> Ударение падает на тот же слог, что и в инфинитиве 

- нажИвший (нажИть)
- налИвший (налИть)
- прожИвший (прожИть)
- понЯвший (понЯть)
- нанЯвшийся (нанЯться)
- начАвший (начАть)
- кормЯщий (кормИть)
- кровоточАщий (кровоточИть)

### С суффиксом -ЕНН-

- облегчЁнный
- ободрЁнный
- обострЁнный
- отключЁнный
- углублЁнный
