# Теория

## Разнообразные простые случаи

| 1 смысл                                                        | 2 смысл                                        | 3 смысл                        |
| -------------------------------------------------------------- | ---------------------------------------------- | ------------------------------ |
| абонемент — документ                                           | абонент — клиент                               |                                |
| адресант — отправитель                                         | адресат — получатель<br>(по пути потерял н)    |                                |
| безответный — не получающий ответа                             | безответственный — не сознающий ответсвенности |                                |
| благотворительный — имеющий целью оказание материальной помощи | благотворный — полезный                        |                                |
| высокий — большая вертикальная протяженность                   | высотный — многоэтажный, очень высокий         |                                |
| гарантрированный — обеспеченный                                | гарантийный — гарантия                         |                                |
| коренастый — крепкого сложения                                 | коренной — исконный, существенный              | корневой — относящийся к корню |
| зачинатель — тот, кто начал что-либо положительное             | зачинщик — тот, кто начал что-то плохое        |                                |
| невежа — невежливый                                            | невежда — малообразованный                     |                                |
| эффектинвый — действенный                                      | эффектный — производящий впечатление           |                                |
| надеть (одежду) — что-то                                       | одеть (Надежду) — кого-то                      |                                |

| С -ЕСК- (связано со словом напрямую)                                        | Без -ЕСК-                                                      |
| --------------------------------------------------------------------------- | -------------------------------------------------------------- |
| артистический — предназначенный для артистов                                | артистичный — искусный, виртуозный                             |
| дипломатический — относящийся к дипломатам                                  | дипломатичный — вежливый                                       |
| демократический строй, государство, республика                              | демократичный образ жизни, решение, руководитель, марка одежды |
| микроскопический — производимый с помощью микроскопа; чрезвычайно маленький | микроскопичный — чрезвычайно маленький                         |
| романтический — относящийся к романтизму; мечтательный, идеальный           | романтичный — мечтательный, идеальный                          |

| каменистый                            | каменный — сделанный из камня                                      |
| лесистый                              | лесной — относящаяся к лесу, живущий в лесу                        |

## Отличие в приставках

- Восполнить — добавить нехватающее (пробел в знаниях, недостаток сил, недостаток магния в организме)
- Пополнить — прибавить нехватающее к имеющемуся (запасы, список)
- Дополнить — добавить к имеющемуся в целостности (сказанное, справочник формулами)
- Заполнить — занять целиком, наполнить; вписать нужные сведения
- Наполнить — заполнить полностью вместилище
- Переполнить — заполнить сверх меры

- Загородить — заслонить, сделать преграду
- Огородить — обнести оградой
- Оградить — защитить
- Отгородить — отделить, поставив перегородку (часть комнаты, двор от улицы)
- Перегородить — разделить пространство на равноправные части (веранду, двор, комнату шкафом)

- Занизить — сделать ниже нормального
- Понизить — сделать более низким; перевести на более низкую должнось
- Снизить — ухудшить, ослабить

- Отличить тодно от другого
- Различить в темнотет
