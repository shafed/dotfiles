```dataviewjs
// имя текущего файла, например «Русский Номер 07»
const fileName = dv.current().file.name;

// последние две цифры номера
const numStr = fileName.slice(-2);
const num    = Number(numStr);

// вычисляем предыдущий и следующий номера (циклически 01 ↔ 27)
const prev = ((num + 25) % 27 + 1).toString().padStart(2, "0");
const next = ((num % 27)  + 1).toString().padStart(2, "0");

// функция-шаблон для meta-bind-button
const btn = (target) => `
\`\`\`meta-bind-button
label:  "Русский Номер ${target}"
style: primary
class: obsidian
actions:
  - type: open
    link: "[[Русский Номер ${target}]]"
\`\`\``;

// выводим кнопки
dv.paragraph(btn(prev) + '\n\n' + btn(next));
```
- Не путай вопросно-ответную форму с риторическим вопросом
- В одной букве может попасться несколько приемов, поэтому придется выделять методом исключения
- По заявлению составителей, риторическое восклицание в дополнение к интонации обязательно сопровождается теми или иными языковыми средствами выражения
	- **О**, это большой честолюбец!
	- В предложении есть нарушение прямого порядка слов (в данном случае подлежащее выражено существительным _«любовь»_ (определение даётся именно этому понятию, а не _«странной вещи»_), следовательно, так как в повествовательных предложениях подлежащее обычно предшествует сказуемому, нужно поменять местами это слово и неделимое словосочетание; кроме того, указательное слово _«это»_, входящее в состав сказуемого, должно располагаться сразу после тире): “_Да, любовь –_ **_это_** _странная вещь!_” **Обратите внимание:** может показаться, что перед нами риторическое восклицание, но нужного дополнительного языкового средства в предложении нет, поскольку _«да»_ в данном случае выражает утверждение (является утвердительной частицей, противоположной по значению отрицательной частице _«нет»_), а не формообразующей, помогающей образовать форму повелительного наклонения.
		- Да, странная **_это_** вещь – любовь!
- Градация бывает странной и иногда ты ее не замечаешь
