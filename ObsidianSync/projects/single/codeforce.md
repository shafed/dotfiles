# [[71A]]

- Чтобы пользоваться string нужно подключить библиотеку `\#include <string>`
- .size() возвращает количество символов в строке
- На C++ нет отрицательного смещения, т.е. для нахождения последнего символа используем `…[.size() - 1]` вместо `…[-1]`

# [[282A]]

`.find("{текст}")` вовзращает индекс подстроки или строки первого вхождения, если ничего не находит, то возвращает -1

string::npos возвращает -1, т.е. означает, что строки нет

# [[734A]]

- `\#include <algorithm>` подключает библиотеку, функции которой выполняют алгоритмы
- `count({начало диапозона}, {конец диапозона}, {искомый элемент})` количество элементов в диапазоне, значения которых соответствуют заданному значению
- `.begin() .end()` указывают на начало и конец строк

# [[339A]]

- `\#include <bits/stdc++.h>` не стандартная библиотека C++ (подключается компиляторами), которая содержит много других библиотека. В данной задаче подгружает `<vector>`, `<iostream>`, `<algorithm>`
- `size_t` — тип переменной, который обозначает размер чего-то, например, массива
- `stoi()` преобразует переменную string в int
- `.substr({индекс},{длина подстроки})` извлекает подстроку из строки
- `sort({начало диапозона},{конец диапазона})` сортирует
- `auto` — тип переменной, который определяется на основании типа переменной справа выражения
- `*{переменная}` разыменовывает переменную, помогает получить доступ к значению, на которое указывает итератор или указатель

# [[118A]]

- `.find_first_of({строка/символ}, {позиция начала поиска})` ищет первое вхождение строки или символа
- `.erase({начало удаления}, {конец удаления})` удаляет подстроку в диапазоне
- `&{переменная}` — адрес переменной
- `tolower()` возвращает символ в нижнем регистре

Для оптимального варианта

- `strchr({строка},{символ})` ищет первое вхождение символа в строке
- если задать символьный тип для переменной, то при вводе строки, строка будет вводиться посимвольно (можно использовать для while(cin>> …))
